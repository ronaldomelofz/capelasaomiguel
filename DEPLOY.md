## Tecnologia

- **localStorage** — dados no navegador (sem Firebase, sem backend)
- **Netlify** — hospedagem estática
- **GitHub** — repositório

## Deploy

1. `git push` para o GitHub
2. Netlify faz deploy automático (se conectado)

Ou manualmente:
```bash
netlify deploy --prod --dir=.
```
